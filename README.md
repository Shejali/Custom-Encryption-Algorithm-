# Custom Encryption Algorithm

## Overview

This repository contains the documentation for a custom encryption algorithm inspired by the classical Atbash Cipher called Jelly Cipher. The project explores how logical and mathematical operations can be combined with the traditional Atbash Cipher to create a stronger encryption method.

The purpose of this project is to demonstrate an understanding of classical cryptography, encryption techniques, and algorithm design.

## Project Contents

* Project Documentation (PDF/DOCX)
* Design and methodology of the custom encryption algorithm
* Explanation of the encryption and decryption process
* Mathematical and logical operations used
* Security analysis and discussion

## Objectives

* Study the Atbash Cipher and its limitations.
* Design a stronger encryption algorithm based on the Atbash Cipher.
* Apply logical and mathematical operations to improve security.
* Demonstrate the encryption and decryption process.
* Evaluate the effectiveness of the proposed algorithm.

## Key Features

* Custom encryption algorithm based on the Atbash Cipher.
* Enhanced using logical and mathematical transformations.
* Reversible encryption and decryption process.
* Educational focus on cryptographic principles and algorithm design.

## Documentation

The repository contains the complete project documentation describing:

* Background of the Atbash Cipher
* Algorithm design
* Encryption process
* Decryption process
* Test Cases
* Critical Evaluation of Jelly Cipher 
* Conclusion 

## Disclaimer

This project was developed for educational and research purposes to demonstrate cryptographic concepts. It is not intended to replace modern industry-standard encryption algorithms for securing sensitive information.
